# מסע האוצר - אפליקציית לימוד חשבון

אפליקציה לתלמידי כיתות א'-ו' המבחינה את רמת הידע בחשבון בצורת משחק.
הילד מלווה על ידי **קפטן תוכי** במסע באיים כדי למצוא אוצר -
כל אי הוא חידה (תרגיל) בדרגת קושי מותאמת.

## מה האפליקציה עושה כיום (גרסה 0.1)

- ✅ מסך פתיחה עם דמות התוכי
- ✅ בחירת אייקון-חיה במקום סיסמה
- ✅ פרופיל ילד: שם, כיתה (א-ו), גיל
- ✅ מפת איים חזותית עם התקדמות מסומנת
- ✅ מסך שאלה עם קלט תשובה חופשי (בלי אופציות בחירה)
- ✅ עידוד אקראי אחרי כל תשובה - גם אם נכונה וגם אם לא
- ✅ כפתור השהיה (האי של התוכי) וכפתור דלג
- ✅ שמירה אוטומטית של כל פעולה - גם של טיוטת התשובה
- ✅ מד-זמן חבוי שמודד זמן אקטיבי בנפרד מזמן השהיה
- ✅ ניתוק זמן ספירה כשהמסך מוסתר/הילד עוזב את הדף

## מה עוד אין (מחכה לשלבים הבאים)

- ⏳ תוכן לימודי אמיתי לפי תכנית הלימודים (יש שאלות-דמה לכל כיתה)
- ⏳ דשבורד המורה - לראות את התוצאות של כל הילדים
- ⏳ חיבור לענן (Firebase) - כדי שהתוצאות יגיעו אלייך מהילדים
- ⏳ התאמת קושי דינמית (אם הילד מצליח - לעלות, אם נכשל - לחזור אחורה)
- ⏳ PWA - להפוך לאפליקציה אמיתית במסך הבית של ההורה
- ⏳ אפקטי קול ומוסיקה

## דרישות מערכת

- Node.js גרסה 20 ומעלה
- npm

## איך מריצים בפעם הראשונה

```bash
npm install
npm run dev
```

ואז פותחים בדפדפן את הכתובת `http://localhost:5173`.

מהטלפון, אם הם באותה רשת WiFi, אפשר להיכנס לכתובת ה-Network
שמופיעה בטרמינל (לדוגמה `http://10.0.0.11:5173`).


## הערות לפיתוח עתידי

### עדכון מאגר השאלות

נמצא בקובץ `src/data/questions.ts`.
כל שאלה צריכה:
- `id` ייחודי
- `text` - נוסח השאלה
- `answer` - תשובה נכונה (כמחרוזת, יכולה להיות שבר "3/4")
- `topic` - נושא (חיבור, חיסור, וכו')
- `difficulty` - דרגת קושי 1-6
- `grade` - הכיתה שאליה משתייכת השאלה
- `hint` (אופציונלי) - רמז

### חיבור לענן (Firebase) - לעתיד

כשהמורה תרצה לראות את התוצאות:
1. ליצור פרויקט Firebase חינמי
2. להוסיף הגדרות Auth + Firestore
3. לעדכן את `src/storage/storage.ts` שיסנכרן את ה-localStorage עם Firestore
4. ליצור מסך נפרד `/teacher` עם דשבורד מוגן בסיסמה
