# תוכניות לימודים - מקור

תיקייה זו מיועדת ל**קבצי המקור** של תוכניות הלימודים (PDFs, מסמכי Word, וכו').
הקבצים כאן הם רק לעיון - הם לא נטענים אוטומטית לאפליקציה.

## איך להוסיף תוכנית לימודים

1. שמרי כאן את ה-PDF של תוכנית הלימודים, בשם בפורמט:
   - `grade1.pdf` - כיתה א'
   - `grade2.pdf` - כיתה ב'
   - וכו' עד `grade6.pdf`
2. הודיעי לי ואני אקרא את התוכן ואחלץ ממנו את הנושאים.
3. אני אבנה את **מאגר השאלות** בקובץ נפרד תחת `src/data/curriculum/grade1.ts` (וכו').
